GitCode CLI · gh-style workflows for GitCode

One CLI for auth, repos, PRs, issues, releases, and raw API access.

GitCode CLI keeps the command layout familiar if you already use gh, while staying focused on GitCode’s REST API and day-to-day repository workflows.

gc --help
$ gc --help

Usage: gc [OPTIONS] COMMAND [ARGS]...

GitCode CLI

Options:
  --profile TEXT              Config profile to use.
  --host TEXT                 Override the GitCode API host.
  --install-completion       Install completion for the current shell.
  --show-completion          Show completion for the current shell.
  --help                      Show this message and exit.

Commands:
  api         Raw GitCode API access
  auth        Authenticate gc with GitCode
  repo        Manage repositories
  pr          Manage pull requests
  issue       Manage issues
  release     Manage releases
  search      Search bundled API docs and GitCode resources
  config      Manage gc configuration
  completion  Generate shell completions
  version     Print gc version
